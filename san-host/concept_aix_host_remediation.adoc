= AIX 호스트 수정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
7-Mode 전환 툴(7MTT)을 사용하여 SAN 환경에서 7-Mode로 운영되는 Data ONTAP을 clustered Data ONTAP으로 전환하는 경우, 전환 합병증 없이 전환 전후에 AIX 호스트에서 일련의 단계를 수행해야 합니다.
