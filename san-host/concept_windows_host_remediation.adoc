= Windows 호스트 문제 해결
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
7MTT(7-Mode 전환 툴)를 사용하여 7-Mode에서 운영되는 Data ONTAP을 clustered Data ONTAP으로 전환하려는 경우 Windows 호스트 전환을 위해 특정 단계를 수행해야 합니다. 또한 컷오버 단계를 준비하기 위한 특정 단계를 수행해야 하며, 전환 후에는 Windows 호스트를 온라인 상태로 전환해야 합니다.
