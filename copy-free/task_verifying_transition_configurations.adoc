= 전환된 구성 확인
:icons: font
:imagesdir: ../media/


[role="lead"]
7-Mode 볼륨 및 구성을 성공적으로 가져온 후에는 전환된 애그리게이트, 볼륨, LUN 및 구성을 수동으로 확인해야 합니다.

.단계
. 7-Mode 애그리게이트, 볼륨, LUN과 CIFS 공유, NFS 엑스포트, LUN 매핑은 전환되었는지 확인
. 7-Mode 구성이 모두 보존되는지 확인합니다.
