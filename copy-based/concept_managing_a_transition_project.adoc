= 전환 프로젝트 관리
:icons: font
:imagesdir: ../media/


[role="lead"]
7-Mode 전환 툴(웹 인터페이스)을 사용하여 전환 프로젝트를 관리할 수 있습니다. 편집, 일시 중지, 다시 시작, 중단, 삭제, 프로젝트에 대해 사전 점검을 실행합니다.
